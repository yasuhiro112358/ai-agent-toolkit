# 解説: なぜ特定のAIサービスに依存しないか

生成 AI のサービスは急速に変化する。特定サービスへの依存は、サービス仕様変更・料金改定・廃止のリスクを抱える。

## 移植性を確保するための制約

- API・外部ツール不使用（プロンプトを貼り付けるだけで動作する）
- Mermaid 等の特定ツール形式に依存しない出力
- Claude / ChatGPT / Gemini 等の主要 UI で同等の結果が得られること

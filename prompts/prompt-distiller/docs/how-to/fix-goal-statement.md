# ハウツー: ゴール記述を修正する

[ドキュメント](../index.md) > ゴール記述を修正する

> **Diataxis: How-to** — ゴールが「何をしたか」に留まっている場合の修正手順。


## 症状

ゴールが「〇〇を要約する」「コードをレビューする」のような動詞句になっている。

## 修正

「なぜその作業が必要だったか」を問いかけ、「〜するために〜する」形式に書き換える。

```text
悪い例: コードをレビューする
良い例: レビューコストを下げるために、コードの問題点を網羅的に列挙する
```

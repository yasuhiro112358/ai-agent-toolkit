# ハウツー: 暗黙の制約条件を補足する

[ドキュメント](../index.md) > 暗黙の制約条件を補足する

> **Diataxis: How-to** — 制約条件に暗黙のルールが抜けている場合の修正手順。


## 症状

明示的に指定したルールだけが書かれており、会話の文脈から自明だった条件が抜けている。

## 修正

会話を振り返り、「当然の前提」として扱っていた条件を補足する。よくある抜け漏れ:

- 出力言語（日本語で出力すること、など）
- トーン（敬語・箇条書き・Markdown 形式、など）
- 禁止事項（固有名詞を使わない、独自の結論を加えない、など）

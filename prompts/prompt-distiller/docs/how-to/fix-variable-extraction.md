# ハウツー: 変数化を修正する

[ドキュメント](../index.md) > 変数化を修正する

> **Diataxis: How-to** — 変数化が不十分な場合の修正手順。


## 症状

要件定義書に特定のプロジェクト名・ファイル名・固有の数値が残っている。

## 修正

固有値を `[変数名]` 形式のプレースホルダーに置き換え、変数一覧に追加する。

```text
変更前: target_audience は 30〜40代の会社員
変更後: target_audience は [対象読者層]

変数一覧に追加:
| `[対象読者層]` | プロンプトを適用する読者の属性（年代・職業等） |
```

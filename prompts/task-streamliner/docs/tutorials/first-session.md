# チュートリアル: 初めてのセッション

`templates/tasks.md` を使って最初のタスク管理セッションを体験する手順。

## 前提

- Claude.ai / ChatGPT 等のチャットUIのアカウント
- テキストエディタ

## 手順

### 1. templates/tasks.md をコピーする

`templates/tasks.md` をテキストエディタで開き、全文を選択してコピーする。

### 2. チャットUIに貼り付けて送信する

チャットUIの入力欄に貼り付けてそのまま送信する。追加の指示は不要。

AIが自動で以下を実行する。

- **Inbox 処理**: Inbox に書いたメモを解析し、サマリータスクとリーフタスクに分解して WBS に追加する
- **CPM 計算**: `depends_on`・`due`・`estimate` からフロートを算出し、WBS テーブルを実行優先順に並べ替える

処理結果が変更サマリーとして返ってくる。

### 3. 自然言語で指示を出す

返ってきたサマリーを確認し、必要であれば指示を出す。

```
「資料作成を始めた」
「レビュー依頼の締め切りを来週金曜に変えて」
「会議準備はキャンセルで」
```

AIは変更サマリーだけを返す（ファイル全文は返さない）。

### 4. templates/tasks.md を保存する

作業が終わったら「出力して」または「終了」と入力する。AIがファイル全文をコードブロックで返すので、それをコピーして `templates/tasks.md` に上書き保存する。

## 次のステップ

- セッションの外で思いついたことは `templates/tasks.md` の `## Inbox` にメモを書いておくだけでよい。次回セッション開始時にAIが自動処理する
- タスクが増えてきたら [アーカイブ](../how-to/archive.md) を参照
